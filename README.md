# S.W.A.T

SWAT is an autonomous AI-IoT stagnant water monitoring & treatment system designed to prevent mosquito breeding and reduce dengue outbreaks. It integrates Arduino-based pH, turbidity, and flow sensors with AI analytics powered by Python, scikit-learn, and Cloudflare Workers AI.

## Features
- Real-time water quality monitoring
- Fixed-threshold detection for pH, turbidity, and flow
- AI trend analysis & risk scoring
- Automated pump activation simulation
- Cloud dashboard via ThingSpeak
- Front-end demo via Streamlit app

## Technologies Used
Python, scikit-learn, Cloudflare Workers AI, Arduino, IoT Sensors, ThingSpeak, Streamlit.
